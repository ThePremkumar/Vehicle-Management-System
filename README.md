# 🚗 Vehicle Management System

A Python-based **Vehicle Management System** designed to streamline the management and tracking of vehicles. This project provides a clean, structured codebase for handling core vehicle operations such as registration, tracking, and record management.

---

## 📁 Project Structure

```
Vehicle-Management-System/
│
├── main.py          # Application entry point
└── src/             # Core source modules
```

---

## ✨ Features

- 🚘 **Vehicle Registration** — Add and manage vehicle records with ease
- 📋 **Vehicle Listing** — View all registered vehicles in a structured format
- 🔍 **Search & Filter** — Quickly find specific vehicles by attributes
- ✏️ **Update Records** — Modify existing vehicle information
- 🗑️ **Delete Entries** — Remove vehicles from the system
- 🗂️ **Modular Architecture** — Clean separation of logic within the `src/` directory

---

## 🛠️ Tech Stack

| Technology | Purpose              |
|------------|----------------------|
| Python 3.x | Core programming language |
| OOP Design | Modular, scalable structure |

## 🚀 Getting Started

### Prerequisites

Make sure you have **Python 3.x** installed on your system.

```bash
python --version
```

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/ThePremkumar/Vehicle-Management-System.git
cd Vehicle-Management-System
```

2. **Run the application**

```bash
python main.py
```

---

## 💡 Usage

Once you run `main.py`, the system will launch and present you with options to:

- Add a new vehicle
- View all vehicles
- Search for a specific vehicle
- Update vehicle details
- Delete a vehicle record

Follow the on-screen prompts to navigate through the system.
